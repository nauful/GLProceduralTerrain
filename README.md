# GLProceduralTerrain
Infinite procedural terrain renderer using OpenGL and GLUT. Tested on Windows and Linux.

## Features
- Geomipmapping
- Procedural terrain generation
- Procedural texture splatting
- Texture synthesis
- Procedural clouds
- Depth of field
